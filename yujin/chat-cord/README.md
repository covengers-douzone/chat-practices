# Real Chat APP with Rooms

## project

* 생성
    ```bash
    npm init
    ```
* 설치
    ```bash
    ## 일반
    npm i express
    npm i socket.io
    npm i moment
    
    ## dev
    npm i -D nodemon
    ```
* scripts (package.json)
    ```json
    "scripts": {
        "start": "node server",
        "dev": "nodemon server"
    },
    ```
